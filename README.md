# Delight_TestTechnique

### How to launch this project

1. Clone the repository with `git clone https://github.com/Yann2808/Delight_TestTechnique.git`
2. Make a copy of `.env.example` and rename it `.env`
3. Fill the `.env` file with your Spotify API keys
4. Install the requirements with `pip install -r requirements.txt`
5. Run the script `main.py`
